## 🔧 Cómo correr el proyecto localmente

```bash
git clone https://github.com/TU_USUARIO/rimac-frontend-challenge.git
cd rimac-frontend-challenge
npm install
npm run dev
```

---

## 📦 Cómo hacer el build

```bash
npm run build
```
